Como Jogar

Objetivo

Encontre todos os pares de cartas iguais. Quanto mais pares você encontrar, mais sucesso!

Como Jogar

  1. Clique ou toque em uma carta para virá-la.
  2. Clique ou toque em outra carta para procurar o par.
  3. Se as duas cartas forem iguais, você encontrou um par!
  4. Continue até encontrar todos os pares.

Dicas

  • Você pode jogar sem pressa - o tempo é ilimitado.
  • Não se preocupe com erros - o jogo não conta erros, apenas sucesso!
  • Escolha a quantidade de pares que for mais confortável para você.
  • Ative o modo calmo para uma experiência ainda mais tranquila.

Controles

  • Clique/Tap: Virar uma carta
  • Tecla Espaço: Virar carta selecionada
  • Teclas de setas: Navegar entre cartas
  • Tecla Tab: Mover entre botões e elementos
